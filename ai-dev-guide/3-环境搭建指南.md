# 环境搭建指南

## 🎯 阶段目标

学习如何搭建完整的 Web 开发环境，为后续的项目开发做好准备。

## 🔍 为什么需要搭建开发环境？

开发环境是进行 Web 开发的基础，包括：
- 代码编辑器：用于编写代码
- 运行环境：用于执行代码
- 包管理工具：用于管理依赖
- 版本控制：用于管理代码变更

## 🛠️ 必备开发工具

### 1. 代码编辑器

**推荐：Visual Studio Code (VS Code)**
- **优点**：免费、轻量、插件丰富、跨平台
- **适用场景**：所有 Web 开发

### 2. 运行环境

**推荐：Node.js**
- **用途**：JavaScript 运行时，用于执行 JavaScript 代码
- **版本建议**：LTS 版本（长期支持版本）

### 3. 包管理工具

**推荐：npm 或 yarn**
- **用途**：管理项目依赖，安装第三方库
- **npm**：Node.js 内置
- **yarn**：Facebook 开发的包管理器

### 4. 版本控制工具

**推荐：Git**
- **用途**：管理代码版本，协作开发
- **关联服务**：GitHub、GitLab、Gitee

**向 AI 提问示例：**
- "如何在 VS Code 中配置终端？"
- "如何设置 npm 镜像源？"
- "如何创建 GitHub 账号并配置 SSH 密钥？"

**配置项目：**
1. **创建项目目录**：
   ```bash
   mkdir my-project
   cd my-project
   ```

2. **初始化 Git 仓库**：
   ```bash
   git init
   ```

3. **创建 .gitignore 文件**：
   ```bash
   echo "node_modules/\n.next/\n.env" > .gitignore
   ```

## 📦 项目初始化

### 使用 create-next-app

**向 AI 提问示例：**
- "如何使用 create-next-app 创建项目？"
- "create-next-app 的参数有哪些？"
- "如何创建包含 TypeScript 的 Next.js 项目？"

**初始化命令：**
```bash
# 创建默认项目
npx create-next-app@latest

# 创建包含 TypeScript、Tailwind CSS 的项目
npx create-next-app@latest --typescript --tailwind
```

### 使用 Vite

**向 AI 提问示例：**
- "如何使用 Vite 创建 React 项目？"
- "Vite 和 create-react-app 有什么区别？"
- "如何使用 Vite 创建 Vue 项目？"

**初始化命令：**
```bash
# 创建 React 项目
npm create vite@latest my-project -- --template react

# 创建 Vue 项目
npm create vite@latest my-project -- --template vue
```
